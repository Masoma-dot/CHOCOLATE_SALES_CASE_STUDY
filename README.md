# CHOCOLATE_SALES_CASE_STUDY
Data analysis project exploring global chocolate sales in 2025. The dataset was cleaned and analysed using Excel spreadsheets, and Tableau to uncover revenue trends, sales performance by country, brand popularity, and purchasing patterns across different sales channels.
